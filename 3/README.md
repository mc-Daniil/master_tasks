## Описание

Сначала при вводе переписать логин на `admin` и пароль на `qwert`, потом переписать указатель на функцию, чтобы вызвалась `secret_func()` и вывела флаг

## Решение

0. Определяем, что перед нами ELF с выключенной митигацией PIE
1. Открываем в IDA Pro. Видим, что есть даже дебаг-символы
![alt text](image.png)
2. В стртоке 3 отключается буферизация для нормальной работы
3. В строке 4 в структуру `user` в поле `password` записывается строка `passs`. Если ЛКМ 2 раза нажать на поле структуры, откроется окно со структурой. `user` - это струткура `Auth`, в которой 3 поля: `name` (16 байт), `password` (16 байт), `action` (указатель на функцию - 8 байт)
![alt text](image-1.png)
4. Далее в поле `user.action` записывается указатель на функцию `greet()`, которая приветствует пользователя по имени
![alt text](image-2.png)
5. Далее в начало структуры `user` (То есть в поле `name`) из пользовательского ввода считывается 64 байта (а под имя выделено всего 16 байт). Значит, можно перезаписать пароль и указатель на функцию
6. После этого вызывается функция, которая в поле `action`
7. Если внимательно посмотреть на список функций слева, можно увидеть функцию `secret_func()`, которая нигде не вызывется и выводит флаг
![alt text](image-3.png)
8. Флаг будет выведен, если имя - `admin`, пароль - `qwert`, а в поле `actioin` - указатель на `secret_func()`
9. На Питоне напишем программу, которая отправить строку формата `admin{10_нуль_байтов}qwert{10_нуль_байтов}{адрес_secret_func()}`
